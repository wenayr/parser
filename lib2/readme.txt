
1. Запуск парсера - скрипт parse в package.json.
Парсит данные с сайта и создаёт итоговый класс в файле result.ts.
А также дополнительно сохраняет структуру данных в файле datas.json на всякий случай.

Файл template.ts - это шаблон будущего документа, из которого затем создаётся result.ts (вставляются фрагменты кода вместо ключей $enums$, $methods$, $weights$)

2. Компиляция - скрипт build в package.json.
Генерируются .js и d.ts файлы в папке lib
